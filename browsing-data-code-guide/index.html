Setup

This guide walks readers through code implementing the analytical steps described in Clemm von Hohenberg, Bernhard, et al. “Analysis of Web Browsing Data: A Guide.” (2023). It is a more extended, and continually updated, version of the Supplementary Materials (Part B) published with the paper.

Data

To run the code, you first need to download a toy data set from our OSF repository at https://osf.io/jmyek and put in your working directory. This is based on a real data set collected from a US sample in 2019. We anonymized the data by deleting (most of) the query strings and paths and excluding rare domains. You can also work with your own data, which should have at least three columns called panelist_id, url and timestamp (in POSIXct format).

R. Load the data set and rename it to distinguish the object we work with in tidy R and with webtrackR.

load("testdt_tracking.rda")
