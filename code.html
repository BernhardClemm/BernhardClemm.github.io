Function for approximating following dates of Twitter followers: The Twitter API does not provide information about when a Twitter user started following another. This Python script allows you to infer the earliest possible date a user can have started following another user. It exploits the fact that Twitter orders the list of followers according to date of following, and provides dates at which a user account was created.

"The role of randomization" (Shiny App): Experiments are considered the "gold standard" of causal inference because through randomization, treatment groups are in expectation equal on known as well unknown variables. The app illustrates this by allowing users to re-assign treatment groups and observe distributions on a number of background variables.
